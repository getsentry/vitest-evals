# vitest-evals Skill

Install the whole `skills/vitest-evals` directory as one skill. Keep
`SKILL.md`, `SPEC.md`, `SOURCES.md`, and `references/` together because runtime
instructions use relative reference paths.

This skill is intentionally focused on the harness-backed `vitest-evals` API:
suite authoring, custom harnesses, first-party AI SDK and Pi AI harnesses,
judges, replay, normalized run data, and targeted verification.
